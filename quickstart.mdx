This guide takes you from zero to a working Hazelcast cluster. You will start a local node, store and retrieve data using `IMap`, run a SQL query, and submit a simple stream processing pipeline.

<Info>
  Hazelcast requires **JDK 17 or later**. SQL features require the `hazelcast-sql` module on the member classpath.
</Info>

## Step 1: Add the dependency

Add Hazelcast to your Java project:

<CodeGroup>

```xml Maven pom.xml
<dependency>
  <groupId>com.hazelcast</groupId>
  <artifactId>hazelcast</artifactId>
  <version>5.5.0</version>
</dependency>
```

```groovy Gradle build.gradle
implementation 'com.hazelcast:hazelcast:5.5.0'
```

```kotlin Gradle (Kotlin) build.gradle.kts
implementation("com.hazelcast:hazelcast:5.5.0")
```

</CodeGroup>

To use SQL queries on a cluster member, also add:

```xml Maven pom.xml
<dependency>
  <groupId>com.hazelcast</groupId>
  <artifactId>hazelcast-sql</artifactId>
  <version>5.5.0</version>
</dependency>
```

## Step 2: Start a Hazelcast instance

Create and start a Hazelcast member. A single node is already a fully functional cluster:

```java QuickStart.java
import com.hazelcast.core.Hazelcast;
import com.hazelcast.core.HazelcastInstance;

public class QuickStart {
    public static void main(String[] args) {
        HazelcastInstance hz = Hazelcast.newHazelcastInstance();
        System.out.println("Hazelcast started: " + hz.getName());
    }
}
```

You should see output like:

```
Members {size:1, ver:1} [
    Member [localhost]:5701 - <uuid> this
]
Hazelcast started: hazelcast_instance_1
```

## Step 3: Store and retrieve data with IMap

`IMap` is the core distributed data structure — a concurrent, partitioned key-value store:

```java IMapExample.java
import com.hazelcast.core.Hazelcast;
import com.hazelcast.core.HazelcastInstance;
import com.hazelcast.map.IMap;
import com.hazelcast.query.Predicates;
import java.util.Collection;

HazelcastInstance hz = Hazelcast.newHazelcastInstance();

// Get (or create) a distributed map
IMap<String, String> capitals = hz.getMap("capitals");

// Store data
capitals.put("France", "Paris");
capitals.put("Germany", "Berlin");
capitals.put("Japan", "Tokyo");

// Retrieve data
String capital = capitals.get("France");
System.out.println("Capital of France: " + capital); // Paris

// Query with a predicate
Collection<String> results = capitals.values(Predicates.equal("__value", "Paris"));
System.out.println("Cities named Paris: " + results); // [Paris]

System.out.println("Total entries: " + capitals.size()); // 3
```

## Step 4: Run a SQL query

Use the SQL service to query your map using standard SQL syntax. First create a mapping that exposes the map as a SQL table:

```java SqlExample.java
import com.hazelcast.core.Hazelcast;
import com.hazelcast.core.HazelcastInstance;
import com.hazelcast.sql.SqlResult;
import com.hazelcast.sql.SqlRow;

HazelcastInstance hz = Hazelcast.newHazelcastInstance();

// Create a SQL mapping for the "capitals" IMap
hz.getSql().executeUpdate(
    "CREATE OR REPLACE MAPPING capitals (" +
    "  __key VARCHAR," +
    "  __value VARCHAR" +
    ") TYPE IMap OPTIONS (" +
    "  'keyFormat' = 'varchar'," +
    "  'valueFormat' = 'varchar'" +
    ")"
);

// Populate data
hz.getMap("capitals").put("France", "Paris");
hz.getMap("capitals").put("Germany", "Berlin");

// Run a SQL SELECT
try (SqlResult result = hz.getSql().execute("SELECT __key, __value FROM capitals ORDER BY __key")) {
    for (SqlRow row : result) {
        System.out.println(row.getObject("__key") + " -> " + row.getObject("__value"));
    }
}
// Output:
// France -> Paris
// Germany -> Berlin
```

## Step 5: Submit a stream processing pipeline

Build a simple batch pipeline using the Jet engine. This example reads from `IMap` and logs each entry:

```java PipelineExample.java
import com.hazelcast.core.Hazelcast;
import com.hazelcast.core.HazelcastInstance;
import com.hazelcast.jet.pipeline.Pipeline;
import com.hazelcast.jet.pipeline.Sources;
import com.hazelcast.jet.pipeline.Sinks;

HazelcastInstance hz = Hazelcast.newHazelcastInstance();

// Populate data
hz.getMap("numbers").put("one", 1);
hz.getMap("numbers").put("two", 2);
hz.getMap("numbers").put("three", 3);

// Build a pipeline: read from IMap, filter, log results
Pipeline pipeline = Pipeline.create();
pipeline.readFrom(Sources.<String, Integer>map("numbers"))
        .filter(entry -> entry.getValue() > 1)
        .writeTo(Sinks.logger());

// Submit the job and wait for it to complete
hz.getJet().newJob(pipeline).join();
```

You should see log output for entries with values greater than 1.
