Hazelcast is a unified real-time data platform that combines a high-performance in-memory data store with a built-in stream processing engine. Use it to process streaming data, enrich it with context, and take instant action — before data ever reaches a database.

Trusted by companies like HSBC, JPMorgan Chase, Lowe's, and Volvo, Hazelcast delivers microsecond latency for key-value lookups and sub-10ms latency at millions of events per second for streaming queries.

## Get started

<Steps>
  <Step title="Add the Maven dependency">
    Add Hazelcast to your Java project using Maven or Gradle.

    ```xml pom.xml
    <dependency>
      <groupId>com.hazelcast</groupId>
      <artifactId>hazelcast</artifactId>
      <version>5.5.0</version>
    </dependency>
    ```
  </Step>
  <Step title="Start a Hazelcast instance">
    Create a local Hazelcast node — it's also a fully functional cluster member.

    ```java
    import com.hazelcast.core.Hazelcast;
    import com.hazelcast.core.HazelcastInstance;

    HazelcastInstance hz = Hazelcast.newHazelcastInstance();
    ```
  </Step>
  <Step title="Store and query data">
    Use `IMap` to put, get, and query distributed data instantly.

    ```java
    import com.hazelcast.map.IMap;

    IMap<String, String> map = hz.getMap("my-map");
    map.put("key1", "value1");
    String value = map.get("key1");
    ```
  </Step>
  <Step title="Run a SQL query">
    Use the SQL service for expressive distributed queries.

    ```java
    try (SqlResult result = hz.getSql().execute("SELECT * FROM \"my-map\"")) {
        for (SqlRow row : result) {
            System.out.println(row.getObject(0));
        }
    }
    ```
  </Step>
</Steps>
